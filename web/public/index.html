How it works

1

Submit

Enter any factual claim. The system computes dual cryptographic fingerprints (BLAKE3 + SHA3-512) and signs it with Ed25519.

2

Verify

A 6-stage Clean Data Engine checks for duplicates (exact, near, semantic), validates the source, and normalizes the format.

3

Certify

The fact receives a Data DNA — an immutable cryptographic certificate with a reliability score. Peer nodes validate via ARC consensus.

Why VARCAVIA

No blockchain overhead

ARC consensus reaches finality in <200ms with reputation-weighted voting. No mining, no staking, no gas fees. Just math.

Defense in depth

Dual hashing (BLAKE3+SHA3), 3-layer deduplication (hash, LSH, semantic), Ed25519 signatures, custody chain. If one layer fails, others catch it.

Runs on a laptop

Full node in 15MB. No GPU, no cloud. Designed for 8GB RAM commodity hardware. Offline-first with peer sync when available.
