How it works

1

Submit

Enter any factual claim. The system computes dual cryptographic fingerprints (BLAKE3 + SHA3-512) and signs it with Ed25519.

2

Verify

A 6-stage Clean Data Engine checks for duplicates (exact, LSH, semantic), validates the source, normalizes format, and computes a score.

3

Certify

The fact receives a Data DNA — an immutable cryptographic certificate. Peer nodes validate via Adaptive Resonance Consensus.

import requests

API = "https://varcavia.com/api/v1"

# Verify a fact
r = requests.get(f"{API}/verify", params={"fact": "Earth has a mean radius of 6371 km"})
d = r.json()
print(f"Status: {d['status']}")            # "verified"
print(f"Tier: {d['trust']['tier']}")         # "T1"
print(f"Authority: {d['trust']['authority_score']}")

# Batch verify
r = requests.post(f"{API}/batch/verify", json={
    "facts": ["Earth has a mean radius of 6371 km", "Water boils at 100 degrees Celsius"]
})
for item in r.json()["results"]:
    print(f"  {item['fact']}: score={item['score']:.2f}")
const API = "https://varcavia.com/api/v1";

console.log(d.trust.tier);           // "T1"
console.log(d.trust.authority_score); // 5

Trust tier system

Facts progress through 5 trust tiers based on attestations, source authority, and time. Higher tiers mean more independent verification.

T0

Unattested

No attestations. Just submitted.

T1

Attested

1+ source, authority ≥1

T2

Cross-checked

2+ domains, authority ≥5

T3

Authoritative

Institutional source, authority ≥15

T4

Canonical

50+ authority, 7+ days, 0 contradictions

Protocol primitives

Data DNA

~250 bytes per certificate

Dual fingerprint (BLAKE3 + SHA3-512), Ed25519 signature, temporal proof with microsecond precision, chain of custody, semantic vector slot.

ARC Consensus

<300ms finality

Adaptive Resonance Consensus with reputation-weighted voting. No mining, no staking, no gas fees. Committee selection by domain competence.

CDE Pipeline

6 validation stages

Hash dedup, LSH near-dedup, semantic dedup, source validation, format normalization, composite scoring. Every fact passes all 6 stages.
